# **App Name**: Coptic Calendar & Weather

## Core Features:

- Current Date Display: Displays the current Gregorian date.
- Coptic Month Conversion: Converts the Gregorian date to the corresponding Coptic month.
- Current Winter Storm: Shows the current winter storm ('Nawa').
- Storm Schedule Table: Presents a schedule of upcoming winter storms without requiring internet access.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), reflecting the depth and wisdom of the calendar's heritage.
- Background color: Light blue (#E3F2FD), provides a serene backdrop.
- Accent color: Gold (#FFC107), adds a touch of tradition and importance.
- Font pairing: 'Literata' (serif) for headlines and titles, lending an air of traditionalism, paired with 'PT Sans' (sans-serif) for body text.
- Use clean, minimalist icons representing weather phenomena and calendar events.
- A clear, information-dense layout to display all relevant data on a single screen.
- Subtle transitions when switching between different calendar views.
